# 基础知识

## 什么是Next.Js

React 开发框架

1. 路由系统
2. 预渲染
3.  自动代码拆分
4. 客户端路由
5. 内置css和Sass支持
6. 快速刷新 - 1
7. Api路由
8. 完全可扩展

## 创建 Next.js 应用程序

```
npx create-next-app nextjs-blog

cd nextjs-blog

npm run dev

// http://localhost:3000
```
